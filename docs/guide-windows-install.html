windows安装教程

下载 MSIX 后，正常 Windows 直接双击安装即可。

1. 正常安装

下载 Codex-Windows-x64.msix 后，直接双击安装。

2. 双击安装失败时，用 PowerShell 安装

如果你的系统 MSIX / AppX 安装链路还在，可以执行：

Add-AppxPackage .\Codex-Windows-x64.msix

3. AppX 服务被裁剪时，解压运行

如果你的 Windows 是企业内部定制版，Store 和 AppX 服务被裁得比较狠，可能无法正常安装。 这种情况下可以把 .msix 改成 .zip。

  1. 把 Codex-Windows-x64.msix 改名为 Codex-Windows-x64.zip。
  2. 解压 zip。
  3. 进入解压后的 app 目录。
  4. 直接运行 Codex.exe。

4. 说明

MSIX 本质上也是包，解压后直接运行通常没有太大兼容性问题。 但正常系统还是建议优先走双击安装或 Add-AppxPackage 安装流程。
